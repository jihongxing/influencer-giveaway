# TabBar 图标说明

## 图标要求

- **尺寸**：81x81px（推荐）
- **格式**：PNG（支持透明背景）
- **颜色**：
  - 未选中：灰色 (#999999)
  - 选中：微信绿 (#07C160)

## 需要的图标

### 1. 首页图标
- `home.png` - 未选中状态
- `home-active.png` - 选中状态
- 建议图标：房子图标

### 2. 发现图标
- `explore.png` - 未选中状态
- `explore-active.png` - 选中状态
- 建议图标：搜索/探索图标

### 3. 创建图标
- `create.png` - 未选中状态
- `create-active.png` - 选中状态
- 建议图标：加号/创建图标

### 4. 我的图标
- `profile.png` - 未选中状态
- `profile-active.png` - 选中状态
- 建议图标：用户/个人图标

## 临时方案

如果暂时没有图标，可以：
1. 使用文字占位（TabBar会自动显示文字）
2. 使用简单的SVG图标
3. 使用在线图标生成工具生成

## 图标资源

推荐使用以下资源：
- [IconFont](https://www.iconfont.cn/) - 阿里巴巴图标库
- [Icons8](https://icons8.com/) - 免费图标库
- [Flaticon](https://www.flaticon.com/) - 免费图标库
