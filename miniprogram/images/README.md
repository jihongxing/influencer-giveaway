# TabBar 图标说明

此目录用于存放小程序 TabBar 所需的图标文件。

## 需要的图标文件

请在此目录下添加以下图标文件（建议尺寸：81x81px）：

1. **首页图标**
   - `home.png` - 未选中状态（灰色）
   - `home-active.png` - 选中状态（绿色 #3cc51f）

2. **我的活动图标**
   - `activity.png` - 未选中状态（灰色）
   - `activity-active.png` - 选中状态（绿色 #3cc51f）

3. **我的订单图标**
   - `order.png` - 未选中状态（灰色）
   - `order-active.png` - 选中状态（绿色 #3cc51f）

## 图标要求

- **格式**: PNG
- **尺寸**: 建议 81x81px（微信小程序推荐尺寸）
- **颜色**:
  - 未选中: #7A7E83（灰色）
  - 选中: #3cc51f（绿色）

## 添加图标后

添加完所有图标文件后，请在 `app.json` 中取消注释 `tabBar` 配置。
